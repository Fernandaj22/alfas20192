Etiquetas de HTML

Las etiquetas HTML son fragmentos de texto rodeados por corchetes angulares < >, que tienen funciones y usos específicos. Se utilizan para escribir código HTML.

Etiquetas generales

<!-- código -->

Etiqueta para escribir comentarios en HTML.

<!DOCTYPE>

Declaración del tipo de documento.

<html>

Marca el inicio y el fin de un documento HTML.

<head>

Etiqueta que define la cabecera de un documento HTML.

<body>

Etiqueta que define el cuerpo de un documento HTML.

Etiqueta head

<title>

Indica el título de la página.

<meta>

Representa metadatos que no pueden ser representados por otros elementos.

<link>

Realiza un enlace entre la página y un elemento o recurso externo.

<script>

Etiqueta para insertar un script.

Secciones de contenido

<h1><h2><h3><h4><h5><h6>

Representan seis niveles de encabezados.

<nav>

Define una grupo de enlaces de navegación.

<footer>

Define el pie de página.

Contenido de texto

<div>

Crea un bloque.

<li>

Define un elemento de una lista.

<ol>

Define una lista ordenada numéricamente o alfabéticamente.

<p>

Representa un párrafo.

<ul>

Define una lista.

Semántica de texto en línea

<a>

Etiqueta para incluir enlaces (links) hacia otras páginas o documentos.

<b>

Convierte el texto a negritas.

<br>

Indica una salto de línea.

<u>

Definir texto subrayado.

Semántica de texto en línea

<img>

Etiqueta para insertar imágenes.

<audio>

Insertar audio.

<video>

Insertar un video.

Contenido de la tabla

<table>

Inserta una tabla en la página.

<thead>

Define el encabezado de una tabla.

<tbody>

Define el cuerpo de una tabla.

<td>

Agregar una columna en una tabla.

<th>

Agrega una "columa encabezado" a una tabla.

Formularios

<form>

Para definir un formulario.

<button>

Crear un botón en donde se puede hacer clic.

<input>

Añade un cuadro de entrada de texto a un formulario.

<select>

Añade un cuadro de entrada de texto a un formulario.
